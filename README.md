# Northstar Issues & Discussions

Welcome! This repo is the public home for:
- Bug reports
- Feature requests
- Questions / troubleshooting
- Product discussions

## Quick links
- **Report a bug:** open an Issue using the Bug Report template
- **Request a feature:** open an Issue using the Feature Request template
- **Ask a question / start a discussion:** use Discussions

## What to include in bug reports
- What you expected vs what happened
- Steps to reproduce
- Screenshots/video (if relevant)
- Your macOS/iOS version + Northstar version
- Any logs or error messages

## Security
Please **do not** post secrets (API keys, JWTs, private keys) or sensitive data.
If you found a security issue, email the maintainers instead of opening a public issue.

## Maintainer notes
- We may convert Discussions ↔ Issues as needed.
- If we close an issue, it’s usually because it’s fixed, duplicated, or needs more info.
